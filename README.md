# Toronto Blue Jays Homebase Challenge

A small web app built for the Toronto Blue Jays developer exercise.

## Overview

This project displays baseball data.

## Features

- Division standings
- MLB news stories
- Team standings
- Player stat pages
- Leaderboard

## Tech Stack

- Python
- Flask
- Jinja
- Tailwind CSS

## Getting Started

### 1. Clone the repo

```bash
git clone
cd
```

### 2. Install dependencies

```bash
uv sync
```

### 3. Run the app

```bash
uv run flask --app app run
```
